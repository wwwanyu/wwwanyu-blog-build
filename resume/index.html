Infocast

2016 — 2017

我在這家叫做 Infocast 的公司的職位沒有被明確定義，有時候參與 wireframe 有時候做行銷素材，有時候參與前端開發。

主要是要翻修原名叫 Elite 的 web app，這是一個收集資料的功能網站，也是這家公司的主力產品。功能是要讓內部分析師可以收集大量的社交平台上的資訊以及新聞，並加上自訂的關鍵字組合，這種組合要可以很複雜。Elite 最後還要能夠讓分析師從收集到的資料，在這個 web app 上製作成報告發送 email 給客戶。

但是我得誠實的說，我當時的能力遠低於這個專案所需要的，所以身心靈鼻青臉腫。
不過我至少在這裡學會 GIT 基本操作以及學到行銷網站與功能網站的差別。
